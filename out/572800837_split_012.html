The Linux Desktop
 
Now that your Linux system has been installed or you have booted into the LiveDVD mode we will examine the desktop environment.  Even though this is a book about the Linux command line, most Linux distributions come with a graphical user interface (GUI) desktop that you will need to navigate in order to get to the command line interface (CLI).
     The following desktop versions of Linux will be covered in this chapter:
     Ubuntu Desktop
     Debian Desktop
     Fedora Desktop
     Mint Desktop
     This is not a complete list of every version of Linux but rather a listing of the most popular distributions.  Some of the Linux distributions, such as Ubuntu, Debian, and Fedora have server versions that do not some with a GUI and all configurations are done on the CLI.
 
Ubuntu Desktop
 
Ubuntu Unity desktop right after the first boot.  Unity is the name of the default Ubuntu desktop manager.  There are other desktop managers that can be used with Ubuntu, but Unity is the most popular.
     When logging into Ubuntu for the first time you will be presented with a Keyboard Shortcuts window.   Clicking the X in the upper left hand corner of the Keyboard Shortcuts window will close the window.  If you want to see the Keyboard Shortcuts again just hold the Super button, also know as the Window button, for 2-3 seconds.  The Super button is normally found at the bottom of the keyboard on the left hand side to the left of the Alt key.  
     The Ubuntu Desktop - Keyboard Shortcuts
     One shortcut that is missing from the Keyboard Shortcuts menu is Ctrl+Alt+T which will open up a terminal window.
